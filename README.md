# 나만의 포트폴리오

## 프로젝트 개요
외부 라이브러리 없이 순수 HTML/CSS/JavaScript만으로 반응형 포트폴리오 웹사이트를 완성합니다. 그 과정에서 DOM 조작, 이벤트, 비동기 등의 개념을 이해하는 것을 목표로 합니다.
<br>
<br>

## 사용 기술
- HTML
- CSS
- JavaScript
- Github API
- Github Pages
